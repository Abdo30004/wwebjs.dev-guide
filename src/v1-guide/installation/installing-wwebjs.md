# Installing WWebJS

<code-group>
<code-block title="npm" active>
```bash
npm init
```
</code-block>

<code-block title="yarn">
```bash
yarn init
```
</code-block>

<code-block title="pnpm">
```bash
pnpm init
```
</code-block>
</code-group>

<code-group>
<code-block title="npm" active>
```bash
npm insall whatsapp-web.js
```
</code-block>

<code-block title="yarn">
```bash
yarn add whatsapp-web.js
```
</code-block>

<code-block title="pnpm">
```bash
pnpm add whatsapp-web.js
```
</code-block>
</code-group>
